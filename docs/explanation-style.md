# סגנון הסברים — "זינוק מחוננים"

מדריך מחייב לכל `explanation` שנוצר במחוללים. מטרת ההסבר: **ללמד את שיטת העבודה**,
לא רק לומר מה התשובה. ההסבר מוצג ב-`whitespace-pre-line` — טקסט בלבד, `\n` נשמר,
אין Markdown. קהל: ילד/ה בכיתה ב' שקורא/ת לבד (הורה לצידו לפעמים).

## התבנית (5 חלקים, לפי סדר)

```
🔍 השיטה: <משפט אחד שמנסח את דרך העבודה הכללית לסוג השאלה>
<גילוי הכלל/הקשר — איך מגלים, לא רק מה הכלל. בחשבון: "שלב 1 — ... שלב 2 — ..."
 עם תרגיל מרוכז בכל שלב ומסקנת ביניים ("מצאנו ש...")>
✓ בדיקה: <אימות הכלל על דוגמה שנייה / הצבת התשובה חזרה — כשרלוונטי>
⚠️ המלכודת: <המסיח המפתה ולמה הוא מפתה — בשם קטגוריית הטעות>
לכן התשובה: <התשובה> ✔
```

לא כל חלק חובה בכל שאלה — אבל "השיטה" ו"לכן התשובה" תמיד, ו"המלכודת"
בכל שאלה שבה המחולל בונה מסיח מכוון (אם המחולל יודע איזה מסיח מפתה — חובה לנצל את זה).

## עקרונות (מזוקקים מסגנון הפסיכומטרי שלנו + התבנית הפדגוגית של חוברות הייחוס)

1. **שיטה קודם, ניבוי לפני הצצה** — ההסבר פותח באיך ניגשים ("קודם מנסחים את הקשר
   במשפט, ורק אז מסתכלים בתשובות"), כדי שהילד יפנים הרגל, לא תשובה.
2. **משפט-גשר ביחסי מילים** — מנסחים את הקשר כמשפט מפורש: "מברשת היא כלי שאיתו
   מציירים" ואז מציבים את זוג-התשובה באותו משפט בדיוק.
3. **שלבים ממוספרים בחשבון** — כל שלב = תרגיל אחד + מסקנת ביניים. לא מדלגים על
   שלב הגילוי ("איך יודעים שההפרש קבוע? בודקים: 7−4=3, 10−7=3 — אותו הפרש!").
4. **אימות ✓** — בצורות-עם-מספרים: מגלים את הכלל מהדוגמה השלמה, מאמתים על עוד
   דוגמה, ורק אז מיישמים על החסר.
5. **פסק-דין נחרץ + סיבה אחת** — בפסילת מסיח: קצר, חד, עם שם לקטגוריית הטעות:
   "כיוון הפוך", "כמעט נכון — קיים רק כלל אחד מהשניים", "תוצאת ביניים",
   "מסיח מראה", "מידע שלא קשור לשאלה".
6. **טון** — גוף ראשון רבים ("נבדוק", "מצאנו"), משפטים קצרים, שאלה מנחה רטורית
   מותרת ("מה עושים קודם? מחפשים מה חוזר על עצמו"). חגיגה קטנה בסוף מותרת.
7. **אורך** — 3–7 שורות. קצר משורה = לא מלמד; ארוך מ-8 = מעייף ילד.
8. **טיפ/שימו לב (אופציונלי)** — שורת "💡 טיפ:" לקיצור חישובי (חוק פילוג, עיגול
   לעשרת) או "שימו לב:" לידע סמוי (שנה = 12 חודשים).

## דוגמאות יעד (מקוריות)

**סדרה חשבונית (במקום "ההפרש: 3. אחרון+3=13"):**
```
🔍 השיטה: בסדרה בודקים קודם מה קורה בין כל שני שכנים.
4→7: הוספנו 3. 7→10: שוב 3. מצאנו את הכלל — כל פעם מוסיפים 3!
✓ בדיקה: 10+3=13, מתאים להמשך.
⚠️ המלכודת: 12 נראה קרוב — אבל הוא נכון רק אם מוסיפים 2, וזה לא הכלל שגילינו.
לכן התשובה: 13 ✔
```

**יחסי מילים (במקום "הקשר: כלי ופעולה"):**
```
🔍 השיטה: הופכים את הזוג למשפט — ואז מציבים בו את התשובות.
המשפט שלנו: "מספריים הם כלי שאיתו גוזרים".
נציב: "מטאטא הוא כלי שאיתו מטאטאים" — מתאים בול!
⚠️ המלכודת: "מטאטא–רצפה" גם מרגיש קשור, אבל זה כלי והמקום שלו — לא כלי והפעולה.
לכן התשובה: מטאטא–לטאטא ✔
```

## גבול תוכן

הסגנון בהשראת החומרים שניתחנו; **כל ניסוח, דוגמה ותרגיל — מקוריים שלנו בלבד.**
אין להעתיק משפטי הסבר, דוגמאות או ניסוחים מחוברות של מתחרים.
